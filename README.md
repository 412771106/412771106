# Python 進階特色彈珠台遊戲（Advanced Pinball Game）

## 一、專題簡介
本專題利用 Python 與 Pygame 框架開發一款具有現代物理特性與關卡要素的彈珠台小遊戲。本版本特別針對「物理參數」進行深度調優，模擬出具備輕盈感的慢速重力模式，並透過自定義的碰撞演算法解決傳統 Pygame 開發中常見的物件穿透問題。

---

## 二、使用套件 / 技術
- **Python 3.12**：核心開發語言  
- **Pygame**：負責圖形渲染、事件處理與遊戲迴圈  
- **Rect Collision**：使用矩形碰撞邏輯，確保精確的物理偵測  
- **Random**：實作發射角度與得分區域移動的隨機性  

---

## 三、遊戲玩法說明
玩家控制下方的綠色彈板，使彈珠持續在場內彈跳。  
彈珠必須擊中畫面上方的移動得分區或特殊要素以累積分數。  
若彈珠掉出底部的發射線，該回合即結束，玩家可隨時按下 **R** 鍵重新開始。

---

## 四、特色玩法
1. **慢速物理系統**  
   - 採用低重力係數（0.25）與固定彈力比  
   - 彈珠動向緩慢且可預測，提升操作手感  

2. **雙向傳送門 (Portal)**  
   - 地圖兩側設有紫色傳送區域  
   - 具備「冷卻機制」，進入一側會由另一側噴出  
   - 增加彈珠路徑多變性  

3. **動態移動得分區**  
   - 藍色目標區會左右持續擺動  
   - 挑戰玩家對彈道角度的判斷力  

4. **強力加速板**  
   - 擊中黃色區域會賦予彈珠瞬間向上的衝量  
   - 可快速回到頂部得分區  

---

## 五、操作方式
| 鍵盤 | 功能 |
|------|------|
| ← / → | 控制綠色彈板左右移動 |
| Space | 啟動遊戲 / 發射彈珠 |
| R | 重置分數並重新開始遊戲 |
