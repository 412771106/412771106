# Python 特色彈珠台遊戲（Pinball Game）

## 一、專題簡介
本專題以 Python 與 Pygame 開發一款經典彈珠台小遊戲，
結合物理碰撞與多媒體元素，並加入特色玩法提升遊戲互動性與挑戰性。

## 二、使用套件 / 技術
- Python 3.10
- Pygame
- math（物理計算）
- random（隨機事件）

## 三、遊戲玩法說明
玩家控制彈珠發射角度與力道，使彈珠在場內彈跳，
擊中目標以獲得分數，彈珠掉落即結束一回合。

## 四、特色玩法
1. **動態重力系統**：隨分數提高，重力與彈力逐漸增加
2. **特殊彈跳區域**：命中後觸發加速或分數加倍效果

## 五、操作方式
- ← → ：調整發射方向
- Space：發射彈珠
- R：重新開始遊戲
